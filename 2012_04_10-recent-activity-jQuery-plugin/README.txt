== What is this? == 

For the Google+ Platform Office hours on 11 April 2012 we walked you through
using the public activity API to fetch a your most recent public activity on
Google+ and render it on your web page. By popular demand we implemented this
as a jQuery plugin.

You can learn more about our office hours here: 
  https://developers.google.com/+/office-hours

== What does it use? ==

 - The JavaScript Google API Client Library: 
     http://code.google.com/p/google-api-javascript-client/
 - jQuery: http://jquery.com/
